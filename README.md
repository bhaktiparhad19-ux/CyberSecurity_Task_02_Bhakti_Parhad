# Cyber Security Task 02: CIA Triad & Risk Assessment

## Objective

The objective of this task is to understand the core principles of cybersecurity using the CIA Triad and learn how to identify, assess, and manage security risks within an organization.

---

## Part A: Understanding the CIA Triad

This section explains the three fundamental principles of information security:

* **Confidentiality** – Protecting sensitive information from unauthorized access.
* **Integrity** – Ensuring data remains accurate, complete, and unaltered.
* **Availability** – Ensuring systems and data are accessible to authorized users whenever needed.

A comparison table is also included to highlight the differences between these three principles.

## Part B: Real-World Case Study

**Selected Case Study:** Facebook Data Leak

## Part D: Threat Identification

This section explains the following common cybersecurity threats:

* Insider Threat
* Malware
* Phishing
* Weak Passwords
* Unpatched Systems

For each threat, the report includes:

* Description
* Possible impact
* Prevention methods

## Part E: Security Recommendations

Based on the identified risks, the following security recommendations were provided:

* Enable Multi-Factor Authentication (MFA)
* Regular system updates and patch management
* Strong password policies
* Security awareness training
* Regular data backups
* Antivirus and endpoint protection
* Role-Based Access Control (RBAC)
* Firewalls and network monitoring
* Security audits and vulnerability assessments
* Incident response planning

Each recommendation includes a brief explanation of its importance.
